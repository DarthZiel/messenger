# Запуск бэкенда

1. Перейдите в папку бэкенда:
   ```bash
   cd backend

2. Активируйте виртуальную среду и установите зависимости:

  ```bash
  poetry shell
  poetry install
```


3. Запустите приложение:
   ```
   poetry run uvicorn main:app --reload

   ```

4. Доступ к админке: Откройте http://127.0.0.1:8000/docs.
